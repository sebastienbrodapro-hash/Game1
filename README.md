# Atelier Stellaire

Jeu web incremental statique en HTML, CSS et JavaScript vanilla.

## Lancer depuis Git

```bash
git clone https://github.com/sebastienbrodapro-hash/Game1.git
cd Game1
```

Ouvrir ensuite `index.html` dans le navigateur.

Aucun serveur local et aucune dependance ne sont necessaires.

## Boucle de jeu

- Produire des etincelles depuis le noyau.
- Acheter des machines pour augmenter la production passive.
- Debloquer des ameliorations et accomplir des commandes.
- La sauvegarde est automatique dans le navigateur.

## Structure

```text
index.html
styles.css
game.js

```
